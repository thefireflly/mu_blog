刚体定轴转动

思路

由于大部分人学过关于直线运动的内容，所以学习转动的时候可以将转动里面的概念类比到直线运动里面，速度类比角速度，加速度类比角加速度。

转动惯量J

在一开始引入转动惯量这个变量的时候，由于并没有解释转动惯量的具体作用，导致在上到这一部分的时候，很多同学会脱节。
这一概念主要用于解释角加速度和力的大小的关系，由于角加速度与合外力矩成正比，且比值与物体本身有关，因此引入转动惯量，用于描述合外力矩与角加速度的比值。转动惯量定义式$ M = J \alpha$，其中$M$为力矩，$\alpha$为角加速度，$J$为转动惯量。
$ J = \sum_i \triangle m_i r_i^2$

转动惯量的推导

根据牛顿第二定律对$J$的值推导
$F_i = \triangle m_i a_i = \triangle m_i \frac{dv}{dt} = \triangle m_i r_i \frac{dw}{dt} = \triangle m_i r_i \alpha$
由于不同的点到轴的距离不同，在等式两端乘$r_i$把力转换力矩。
$ r_i F_i = \triangle m_i r_i^2 \alpha $
合外力矩为$M = \sum_i r_i F_i$
所以$ M = \sum_i \triangle m_i r_i^2 \alpha$，$\alpha$为定值，将$\alpha$提出，得到等式：
$ J = \sum_i \triangle m_i r_i^2 $

角动量

类比于直线运动中的动量，角动量$ L = J w $角动量的变化率$M = \frac{dL}{dt} = 0$

功和能

从a位移到b所做的功为$ A_{ab} = \int_b^a F dl = \int_b^a F r d\theta = \int_{\theta_b}^{\theta_a} Md\theta$
计算动能
$ \int_{\theta_b}^{\theta_a} Md\theta = \int_{\theta_b}^{\theta_a} J\alpha d\theta = J \int_{\theta_b}^{\theta_a} \frac{dw}{dt} d\theta = \int_{w_b}^{w_a} wdw = \frac{1}{2}Jw_a^2 - \frac{1}{2}Jw_b^2 $
则转动动能为$ E_k = \frac{1}{2} Jw^2 $
