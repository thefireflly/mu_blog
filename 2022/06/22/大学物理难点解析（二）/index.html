大学物理难点解析（二）

发表于 2022-06-22
温度与气体动理论,热力学第一定律

本章的知识点分布比较零散，也比较抽象，其中很多知识点不便于理解记忆。也许我们可以尝试在某种程度上将这些公式完全分割开来记忆。

理想气体温标

这里简单的介绍了一下玻意耳定律，由于在高中大部分人已经学过了这方面的知识，这里简单叙述一下。对于一定质量的同种理想气体，$ \frac{pV}{T} = 常数 $，其中$ k $是定值。
大学物理对$k$的具体值进行了更深入的分析，对于不同质量的相同气体，$k$的值和摩尔数成正比，所以相同气体之间可以通过摩尔数来计算。$V_0$，$p_0$，$T_0$分别表示在单位摩尔的下的某种气体的状态参量，$\gamma$表示物质的量，得到公式：
$ pV = \gamma \frac{p_0 V_0}{T_0} T$
阿伏伽德罗定律指出在相同的温度和压强下，一摩尔理想气体的体积相同，所以$ \frac{p_0 V_0}{T_0} $是一个常量，我们称为普适气体常量。
$R \equiv \frac{p_0 V_0}{T_0} = 8.31(J/(mol \cdot K)) $
$ pV = \gamma R T $
这就是最重要的公式之一，称为理想气体状态方程，有很多公式在理想气体状态方程上衍生出来：
$ pV = \frac{m}{M} RT $
$ N_A = 6.023 \times 10^{23} / mol $
$ k \equiv \frac{R}{N_A} = 1.38 \times 10^{-23} J/K $
$ pV = NkT $
第一个公式是用气体质量除以摩尔质量算出摩尔数，第二个是阿伏伽德罗常数，每个摩尔所包含的分子数。基于阿伏伽德罗常数推导出玻尔兹曼常数$k$，下面是把理想气体状态方程的普适气体常量改为玻尔兹曼常数。
这些方程原理比较简单，但是在后面的推导过程中会用到，因此需要尽量记忆。

准静态过程

准静态过程的介绍比较抽象，导致很多同学不理解，其实这一部分的主要目的是解释一些热机循环的前置条件，为的是保证在热机循环的过程中热机充分对外做功。
这一部分的理解并不影响做题。

热容

热容就是物体吸收的热量和变化量的比值，相同物体在不同的变化过程中热容大小不同。

等压过程

压强不变的情况下，气体随着升温一定是会膨胀的，膨胀会对外界做功，所以气体吸收的一部分热量在对外做功的过程中被消耗掉了。
$(dQ)_p = dE + pdV$
$ (\frac{dQ}{dT})_p = \frac{dE}{dT} + p (\frac{dV}{dT})_p $
本章节我们主要计算的是摩尔热容，也就是单位摩尔的热容，因此
$ C_{p,m} = \frac{1}{\upsilon} (\frac{dQ}{dT})_p = \frac{1}{\upsilon} \frac{dE}{dT} + \frac{p}{\upsilon} (\frac{dV}{dT})_p $
将前面在能量均分定理中学到的$E = \frac{i}{2}\upsilon RT$和$pV = \upsilon RT$代入进去
$ C_{p,m} = \frac{i}{2}R + R $

等体过程

在这一情况下热量吸收的过程中没有对外做功，所以$(dQ)_V = dE$，直接带入$E = \frac{i}{2}\upsilon RT$得到$C_{V,m} = \frac{i}{2}R$

绝热过程

绝热条件下，由于处于准静态过程，所以气体的体积变化会对外界做功影响内能，对外做功的大小等于内能变化大小，两者变化方向相反。列出等式
$dE+dA = 0$
由于是绝热过程，A的变化只有通过气体膨胀（收缩）对外做功（负功），因此
$dA = pdV$
内能公式：$dE = \frac{i}{2}\upsilon RdT$
带入可得$\frac{i}{2}\upsilon RdT + pdV = 0$
再计算理想气体状态方程$pV = \upsilon RT$
这里把由于$\upsilon R$都是常数，把这个看成T关于p和V的二元函数，求微分可得
$pdV + Vdp = \upsilon RdT$
把上面计算得到的两个等式联立得到
$(i+2)pdV + iVdp = 0$
$\frac{dp}{p} + \frac{i+2}{i} \frac{dV}{V} = 0$
定义比热比$\gamma$使得$\gamma = \frac{i+2}{i} $
$\frac{dp}{p} + \gamma \frac{dV}{V} = 0$
解这个微分方程可得
$lnp + \gamma lnV = C$（$C$为常数）
$pV^{\gamma} = C_1$
这就是在绝热过程中气体与压强变化的方程，称为泊松公式
如果在联立两个方程的时候不消掉dT，还可以选择消掉dp或者dV，两个分别可以得到另外两个公式
$TV^{\gamma - 1} = C_2$
$ p^{\gamma - 1} T^{-\gamma} = C_3 $

等温过程

等温过程的计算过程并不涉及热容，但是我比较倾向与把几个过程放在一起讲，便于后面对于各种循环的理解。
等温过程中温度不变意味着内能不变，所以$ dE = 0 $
与上面一样，由于是准静态过程，气体会对外做功，气体吸收的热量等于对外的做功，$ Q = A $。
$ A = \int^{V_2}_{V_1} pdV $
在这个过程中$p$和$V$是变量，其他的都是定值，所以$p$和$V$的关系满足$ pV = \upsilon RT $
将$ pV = \upsilon RT $带入方程
$ A = \int^{V_2}_{V_1} \frac{\upsilon RT}{V} dV = \upsilon RT \int^{V_2}_{V_1} \frac{dV}{V} = vRT ln \frac{V_2}{V_1} $

特殊过程

热机循环

热机吸热对外做功，做功大小和吸收热量的比值就是效率，因为热机循环的过程中包括了吸热和放热，设吸热为$Q_1$，$Q_2$为放热:
$ A = Q_1 - Q_2 $
$ \eta = \frac{A}{Q_1} = 1 - \frac{Q_2}{Q_1} $

考试范围主要包括空气标准奥托循环和卡诺循环

空气标准奥托循环

空气标准奥托循环过程为：
1、绝热压缩
2、等体吸热
3、绝热膨胀
4、等体放热
由于有两个等体过程，设绝热膨胀从V_1膨胀到V_2，根据公式${T}V^{\gamma - 1} = C_2$，可得当绝热压缩过程中的$ \frac{T_1}{T_2} $的值等于绝热膨胀过程中的$ \frac{T_4}{T_3} $即
$ \frac{T_1}{T_2} = \frac{T_4}{T_3} = \frac{V_2^{\gamma - 1}}{V_1^{\gamma - 1}} $
循环中的吸热过程和放热过程分别是2和4过程，等体过程不对外做功，所以$ Q_1 = \Delta C_{V,m}vRT = C_{V,m}vR(T_3 - T_2) $
$ Q_2 = \Delta C_{V,m}vRT = C_{V,m}vR(T_4 - T_1) $
$ \eta = 1 - \frac{Q_2}{Q_1} = 1 - \frac{T_4 - T_1}{T_3 - T_2} = 1 - \frac{V_2^{\gamma - 1}}{V_1^{\gamma - 1}} $
设$ \frac{V_2}{V_1} $为压缩比$ r $
$ \eta = 1 - \frac{1}{r^{\gamma - 1} } $

卡诺循环

卡诺循环过程为：
1、等温膨胀
2、绝热膨胀
3、等温收缩
4、绝热收缩
其中吸热放热的过程为1，3
$ Q_1 = vRT_1 \cdot ln\frac{V_2}{V_1} $
$ Q_2 = vRT_2 \cdot ln\frac{V_4}{V_3} $
再把过程2，4的关系式列出来
$ \frac{T_2}{T_1} = (\frac{V_2}{V_3})^{\gamma-1} = (\frac{V_1}{V_4})^{\gamma-1} $
直接带入公式：
$ \eta = 1 - \frac{Q_2}{Q_1} = 1 - \frac{T_2ln\frac{V_4}{V_3}}{T_1ln\frac{V_2}{V_1}} $
根据过程2，4可以得到$ \frac{V_2}{V_3} = \frac{V_1}{V_4} $，带入
$ \eta = 1 - \frac{T_2}{T_1} $
